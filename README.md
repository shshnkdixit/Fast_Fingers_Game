# ⌨️ Fast Fingers Game (Python)

A simple command-line typing speed game built in Python that measures typing speed (WPM), accuracy, and mistakes based on user input.

---

## 🚀 Features
- Random sentence generation
- Typing speed measurement (Words Per Minute)
- Accuracy calculation
- Mistake tracking
- Simple and interactive CLI game

---

## 🛠️ Tech Stack
- Python 3
- random module
- time module

---

## 📦 Installation & Setup

### 1. Clone the repository
git clone https://github.com/shshnkdixit/Fast_Fingers_Game.git

### 2. Move into project directory
cd Fast_Fingers_Game

### 3. Run the game
python fast_fingers.py

---

## 🎮 How to Play
1. Run the program
2. Read the displayed sentence carefully
3. Type it as fast and accurately as possible
4. Get your results:
   - ⏱ Time taken
   - ⚡ WPM (Typing Speed)
   - 🎯 Accuracy
   - ❌ Mistakes
